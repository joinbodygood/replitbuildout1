We are migrating from Shopify to this platform. The following data has been exported and needs to be imported. Build import tools in the admin dashboard for each data type, then I will upload the exported files.

1. CUSTOMER LIST (20,790 customers)
Build a customer import tool in the admin dashboard (Settings > Import > Customers) that accepts a JSON file upload.

The JSON file contains an array of customer objects with these fields:

id (Shopify ID — store as shopify_id for reference)
email
first_name
last_name
phone
tags (comma-separated string — preserve these, they contain segment info like #self-pay, #compound, #insurance, quiz-lead, #VIP, etc.)
created_at
orders_count
total_spent
state (enabled/disabled)
Import rules:

Skip customers with no email address
Deduplicate by email — if the same email appears twice, keep the one with more orders
Preserve the original created_at date (don't overwrite with import date)
Store tags in a searchable format so we can segment customers later (e.g., filter all customers tagged #self-pay or quiz-lead)
Do NOT create login credentials for imported customers — they will need to register fresh on the new site. Store them as "imported leads/customers" with their purchase history metadata
Create customer segments based on the Shopify tags:
#self-pay → Self-Pay Customers
#compound → Compound Medication Customers
#insurance → Insurance Customers
#oral → Oral Medication Customers
quiz-lead → Quiz Leads (didn't purchase)
#VIP or Gold VIP → VIP Customers
#vitaminsandsupplements → Supplement Customers
newsletter or Email-Subscriber → Email Subscribers
Show import summary after completion: total imported, skipped (no email), duplicates merged

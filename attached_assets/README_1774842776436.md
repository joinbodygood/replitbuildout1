# Pharmacy Search -- Integration Guide

## Files

| File | Destination in Project |
|------|----------------------|
| `api-route.ts` | `app/api/pharmacy-search/route.ts` |
| `PharmacySearch.tsx` | `components/pharmacy-search/PharmacySearch.tsx` |

## Quick Setup

### 1. Copy the API route

```
app/
  api/
    pharmacy-search/
      route.ts        <-- paste api-route.ts here
```

### 2. Copy the component

```
components/
  pharmacy-search/
    PharmacySearch.tsx  <-- paste PharmacySearch.tsx here
```

### 3. Make sure Poppins + Manrope fonts are loaded

In your root layout (`app/layout.tsx`), both fonts should already be included via `next/font/google` or a `<link>` tag. If not:

```tsx
import { Poppins, Manrope } from "next/font/google";

const poppins = Poppins({
  subsets: ["latin"],
  weight: ["600", "700"],
  variable: "--font-poppins",
});

const manrope = Manrope({
  subsets: ["latin"],
  weight: ["400", "500"],
  variable: "--font-manrope",
});
```

The component references fonts using Tailwind arbitrary values (`font-['Poppins']`, `font-['Manrope']`), so the fonts just need to be available in the page.

## Usage on the Recommendation Page

```tsx

import type { PharmacySelection } from "@/components/pharmacy-search/PharmacySearch";

  const [pharmacy, setPharmacy] = useState<PharmacySelection | null>(null);

  function handlePharmacySelected(selection: PharmacySelection) {
    setPharmacy(selection);
    // Proceed to next step, save to state/context, or submit to DoseSpot
  }

  return (
    <div>
      {/* ... other recommendation page content ... */}

      {/* Show this when user picks "Pharmacy Pickup" */}
      <PharmacySearch onSelect={handlePharmacySelected} />
    </div>
  );
}
```

## PharmacySelection Shape (for DoseSpot)

The `onSelect` callback returns this object, ready for DoseSpot routing:

```typescript
interface PharmacySelection {
  pharmacy_name: string;      // "CVS PHARMACY"
  pharmacy_zip: string;       // "33133"
  pharmacy_phone?: string;    // "(305) 555-1234"
  pharmacy_npi?: string;      // "1234567890"
  pharmacy_address?: string;  // "123 Main St, Miami, FL 33133"
  pharmacy_fax?: string;      // "(305) 555-5678"
}
```

The `pharmacy_npi` is the key field DoseSpot needs to route the prescription.

## API Details

- **Endpoint:** `GET /api/pharmacy-search?zip=33133`
- **Source:** NPI Registry (CMS) -- free, no API key required
- **Results:** Up to 15 pharmacies per zip code, filtered to LOCATION addresses only
- **Caching:** Response is revalidated every hour via Next.js fetch cache

## Notes

- The zip input restricts to numeric characters and 5-digit max on the client side.
